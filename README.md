# Online-Quiz-App
Online Quiz App
Objective: Build a simple console-based quiz application that allows users to answer multiple-choice questions and displays their final score.

Tools:

Java (JDK) or Python

VS Code / IntelliJ / Terminal

Features:

Displays a set of questions with options

Takes user input for answers

Calculates and shows final score

Outcome:
A functional console quiz app to test general knowledge or subject-based questions.
